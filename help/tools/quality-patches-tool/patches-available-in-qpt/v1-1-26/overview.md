# [!DNL Quality Patches Tool] (QPT) v1.1.26 - översikt

Detta underavsnitt innehåller en detaljerad beskrivning av de problem som åtgärdats av de korrigeringar som är tillgängliga i [!DNL Quality Patches Tool] (QPT) v1.1.26.

QPT v1.1.26 innehåller följande patchar:

1. **ACSD-47937**: Korrigerar problemet där meddelanden om prisfall inte alltid skickas på grund av cachelagring på programnivå.
1. **ACSD-47988** (för Adobe Commerce och Magento Open Source >=2.3.7 &lt;2.4.6) - Korrigerar problemet där produktexport trimmar HTML-taggar från produktbeskrivningen i Page Builder.
1. **ACSD-48212** (för Adobe Commerce och Magento Open Source >=2.3.7 &lt;2.4.6) - Korrigerar problemet där produktimporten tilldelar produkten fel källa.
1. **ACSD-48366** (för Adobe Commerce och Magento Open Source >=2.4.4 &lt;2.4.6) - Korrigerar problemet där produktbilden inte visas i e-postmallen Tillbaka till Stock.
1. **ACSD-48587** (för Adobe Commerce och Magento Open Source >=2.3.7 &lt;2.4.7) - Korrigerar problemet där HTML-specialtecken i produktwidgetens matchningsregler förhindrar att de visar matchande produkter.
1. **ACSD-48661** (för Adobe Commerce och Magento Open Source >=2.3.7 &lt;2.4.7) - Korrigerar problemet där företagets kreditgräns är större än 99, där kommaavgränsaren förhindrar att företaget sparas på grund av ett valideringsfel.

Använd menyn till vänster för att navigera till en viss korrigeringssida.
